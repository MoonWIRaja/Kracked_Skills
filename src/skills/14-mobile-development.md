# Skill 14: Mobile Development

## Overview
iOS and Android development using cross-platform and native technologies.

## Tech Stack
- **Cross-Platform**: React Native, Flutter, Expo
- **Native**: Swift (iOS), Kotlin (Android)
- **Hybrid**: Capacitor, Ionic

## Best Practices

### Performance
- Minimize re-renders
- Optimize images
- Lazy load screens

### Offline Support
- Local storage strategies
- Sync when online
- Conflict resolution

### Platform Guidelines
- Follow iOS HIG
- Follow Material Design
- Platform-specific navigation

## When to Apply
- Mobile Developer: All mobile work
- Engineer: Mobile features
- UX Designer: Mobile UI
