# Naming Conventions

## Variables
- camelCase for variables: `userName`, `itemCount`
- SCREAMING_SNAKE_CASE for constants: `MAX_RETRIES`

## Functions
- camelCase: `getUserById`, `calculateTotal`
- Verb prefix for actions: `get`, `set`, `create`, `update`, `delete`

## Classes/Types
- PascalCase: `UserService`, `OrderItem`

## Database
- snake_case for tables: `users`, `order_items`
- snake_case for columns: `created_at`, `user_id`

## Files
- kebab-case for general: `user-profile.tsx`
- PascalCase for components: `Button.tsx`
