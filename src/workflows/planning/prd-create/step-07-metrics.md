# Step 7: Metrics

## Context
Define how we measure success for the product.

## Instructions

### 1. Identify Key Metrics Categories

- **Business Metrics:** Revenue, conversion, retention
- **User Metrics:** Engagement, satisfaction, growth
- **Technical Metrics:** Performance, reliability, quality

### 2. Define KPIs

For each metric:
- Name
- How to measure
- Target value
- Measurement frequency

### 3. Leading vs Lagging

- **Leading indicators:** Predict outcomes (early signals)
- **Lagging indicators:** Confirm outcomes (results)

### 4. Set Baselines and Targets

- Current baseline (if exists)
- Target for MVP
- Target for v1.0
