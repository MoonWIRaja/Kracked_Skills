# Changelog

All notable changes to Kracked_Skills will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [3.0.0] - 2026-02-14

### Added
- **Global Rebranding**: Entire system renamed to **Kracked_Skills**.
- **Full Suite Integration**: 80+ commands integrated from BMAD, covering Strategy, Innovation, QA, and Testing.
- **Senior Full Stack Architect Persona**: Integrated advanced persona with 15+ years experience.
- **Backend Protocols**: Added "Bulletproof Reliability" standards including OWASP security, Zod validation, and performance guidelines.
- **New Commands**: `/KD-kickoff`, `/KD-api-design`, `/KD-refactor`, `/KD-test`.
- **Ultrathink Protocol**: Multi-dimensional analysis trigger for deep architectural reasoning.
- **Windows Uninstaller**: New `uninstall.ps1` script for PowerShell users.
- **One-Liner Maintenance**: Ability to run `update`, `uninstall`, and `validate` via remote one-liners (`irm | iex`).

### Changed
- Rebranding of all files and references from `KRACKED_skill` to `Kracked_Skills`.
- Updated Maintenance documentation in `README.md`.


## [1.0.0] - 2026-02-13

### Added
- Initial release of Kracked_Skills (KD)
- Full KRACKED v3.3 system prompt
- 9 structured roles (Analyst, PM, Architect, Tech Lead, Engineer, QA, Security, DevOps, Release Manager)
- 7 workflow stages (Discovery, Requirements, Architecture, Implementation, Quality, Deployment, Release)
- Multi-agent support: Party Mode and Agent Swarm
- Confidence scoring and conflict resolution
- Dual language support (English & Bahasa Melayu)
- Persistent status tracking via `status.md`
- Decision validation blocks with structured checkpoints
- Adapters for Claude Code, Cursor, and Antigravity
- Generic adapter for other AI tools
- Installation system via `curl | bash` or `wget | bash`
- 9 document templates
- 6 quality checklists
- 4 workflow definitions
- Complete documentation suite
- Example projects
- Test suite
